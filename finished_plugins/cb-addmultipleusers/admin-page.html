Add multiple users in bulk

I created this plugin as a teaching aid so I could set up a class of students very quickly based just on their names.

Instructions:

  • Type the usernames separated by a comma ('bill bloggs' will give a username of 'bill_bloggs').
  • type password text to add to username
  • select WordPress role for users
  • Press 'create users'
  • Note: a false email will be created for each user!

Go to Dashboard > users to edit/delete users.
